# vue-monaco-monorepo

Use [monaco-editor](https://microsoft.github.io/monaco-editor/) loaded in [Vue 2&3](https://vuejs.org/), no need to configure plugins in `webpack` (or `rollup`, ` vite`) and other packaging tools.
